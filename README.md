# geekchat
一个自学了6天JS的菜鸡写的聊天室，看起来还不错，基于Node.js。

安装Node.js后，需要在geekchat目录下用命令提示符执行命令：`npm install`,等待安装完后执行 `node app.js`，即可在本地使用此聊天室
